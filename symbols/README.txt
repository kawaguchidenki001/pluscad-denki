【重要】ファイル名は文字化け防止のため英数字のみです。
日本語の正式名称は各SVGファイル名に対応する symbols.json の "name" フィールドを参照してください。
（例：ファイル 01_lt_ceil.svg → symbols.json 内で name:"照明 天井付"）

【フォルダ構成】
 svg_mono/   個別SVG（黒・実図面向け）
 svg_color/  個別SVG（分類色付き・確認/UI向け）
 symbols.js  定義配列（アプリ組込用）
 symbols.json 同データ（JSON、日本語名称はここに保持）
 symbol_sheet.svg  全記号一覧ベクター
